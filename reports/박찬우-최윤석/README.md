# 게임 컨셉 제안서 

## 게임의 개요
> 여러 visual elements와 상호작용하여 다양한 맵의 퍼즐을 풀거나 몬스터를 처치하여 스테이지를 진행하는 것이 목적이다. <br/>
> 랜덤 요소를 기반으로 캐릭터를 성장 시킬 수 있으나 그 결정은 돌이킬수 없다.

## 장르 
> Roguelike Adventure
